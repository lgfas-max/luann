# Artesô
> *Feito para mulheres que tecem a própria história*

Plataforma digital para conectar artesãs, clientes e gestores do PRODARTE.

## Sobre o Projeto

O artesanato representa **3% do PIB** nacional e movimenta cerca de R$ 100 bilhões, sendo que **77% dos artesãos brasileiros são mulheres** (SEBRAE). Mesmo assim, 43% das artesãs ainda dependem exclusivamente das feiras físicas para vender.

**Problema:** falta de plataforma integrada para gestão do PRODARTE, vendas limitadas ao espaço/dia da feira, e público sem canal único para descobrir produtos.

**Solução:** uma plataforma com três perfis de uso.

## Funcionalidades

### Cliente
- Mapa de feiras próximas
- Catálogo de produtos com storytelling
- Compra online direta

### Artesã
- Painel de vendas e estoque
- Simulador de precificação com margem de lucro
- Acesso a capacitações

### Gestor PRODARTE
- Painel administrativo centralizado
- Indicadores do programa
- Gestão de inscrições e eventos

## Tecnologias Utilizadas

- **Backend**: Python 3.10+
- **Prototipagem**: Figma
- **Site institucional**: Google Sites
